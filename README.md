# AutoRoles & Embed Messages Discord Bot

A simple and powerful Discord bot that automatically assigns roles to new members and sends customizable embed messages with images and titles.  

## Features
- Automatic role assignment for new members
- Welcome messages in Embed format
- Customizable title and image
- Easy to setup and run

## How to run
1. Install dependencies:
```bash
npm install

Deploy commands:
node deploy.js
node index.js
